<<<<<<<<<<<<<<<<< MVP >>>>>>>>>>>>>>>>>

user can login and create an account
user can add/edit/delete prescribed meds
user can add/edit/delete doctors
user can add/edit/delete pharmacies
user can look up all meds
user can look up all doctors
user can look up all pharmaciess

<<<<<<<<<<<<<<< DATABASE >>>>>>>>>>>>>>>

User table:
user_id (PK)    |pk| Integer
user_email      |  | String
user_password   |  | String

Prescription table:
medication_id   |FK| *API id
user_id         |FK| Integer
doctor_id       |FK| Integer
med_dosage      |  | Integer
dose_frequency  |  | Integer
refill_Date     |  | Date

Medication table:
medication_id   |pk| *API id
brand_name      |  | String
generic_name    |  | String
